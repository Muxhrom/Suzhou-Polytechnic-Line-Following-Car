# Suzhou Polytechnic Line-Following Car

苏州工学院电子设计竞赛校内赛项目：基于 51 系列单片机的智能循迹小车。

## 项目简介

本项目使用 IAP15F2K61S2 单片机，实现黑线循迹、红外过门计数、OLED 状态显示和蜂鸣器提示。系统使用 Timer0 产生 1 ms 时间基准，并完成电机 PWM、按键消抖、显示刷新和计时等任务的调度。

主要功能包括：

- 五路红外传感器循迹
- 双电机差速转向与方向记忆
- 红外过门检测与计数
- OLED 运行时间和状态显示
- 蜂鸣器提示及超时保护

## 开发环境

- Keil uVision（可打开 `SzCar/car3.uvproj`）
- IAP15F2K61S2 单片机
- 12 MHz 系统时钟
- 7.4 V 电池及 5 V 稳压供电

## 使用说明

1. 使用 Keil uVision 打开 `SzCar/car3.uvproj`。
2. 根据实际硬件连接检查源码中的引脚定义和电机方向适配。
3. 编译工程并将生成的固件下载到单片机。
4. 在赛道上检查五路传感器逻辑、电机方向和 OLED 显示效果。
